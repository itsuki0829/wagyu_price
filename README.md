# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

リモートリポジトリは、複数人で共有・バックアップをするネットワーク上の保管庫である。
ローカルリポジトリは自身のPC上で作業・コミットする専用の保管庫である。

## プッシュとマージの違いは何でしょうか？

プッシュとは、自分のPCで作成したコミットをリモートリポジトリに共有する
マージとは、ブランチで行った作業を主幹ブランチへ結合する

## コミットとプッシュの違い

コミットはローカルに変更履歴を保存すること
プッシュはその変更をリモートへ送ること

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

何を変更したか一目でわかるように、簡潔に書くこと
## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

ローカルマージは、自分だけでマージする必要する方法
プルリクエストは、他の人にレビューしてもらってからマージする方法

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

どちらの変更を残すか確認して、不要箇所を削除して修正後にコミットする
